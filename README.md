# やりたくないタスク逆提案AI(YAMLファイルから値を引っ張ってるだけ)
**「タスクをこなすより、まずは逃げる理由を考えたい」**
そんな人類の本質的な欲求に寄り添う Web アプリです。

Rails × Docker × YAML の構成で、
「もっともらしい理由」を自動生成する、
**“生産性を下げないための”言い訳支援サービス** となっています。

---

## 🎯 アプリ概要
やりたくないタスクを入力すると、
このアプリが「それっぽい理由」を *AI風に* 提案してくれます。

- やる気が出ない日
- 締め切りが迫らない日
- 理由はないけどとにかくやりたくない日

そんな日常に、ほんの少しの**正当化**を。

生成される言い訳は、
YAMLファイルをもとにランダム組み合わせで作られるため、
**毎回違う“もっともらしい何か”** が返ってきます。

---

## 😎 特徴（真面目だけど少しふざけてる）

### ✔ 「やらない理由」を自動生成
「今日は気圧が…」「宇宙が…」「集中力が…」など
**一見それっぽい**理由を提示します。

### ✔ 代替案まで提案
「散歩でもして気分転換しましょう」など、
**まるでまともなアプリのような顔**もします。

### ✔ 履歴管理機能
過去の言い訳が一覧表示できるため、
**自分がどれだけ逃げ続けてきたか**客観視できます。

### ✔ PC向けの落ち着いた UI
ふざけた機能のわりに、
**UI はしっかり業務アプリっぽく綺麗です。**

### ✔ Docker ですぐ動く
環境構築でつまずかずに、
**本題の“言い訳生成”に集中できます。**

## 📂 主な機能

- タスク入力フォーム
- 言い訳自動生成（AIっぽいランダム合成）
- 過去の言い訳履歴表示
- 個別結果画面
- バリデーション
- “まともなアプリ風のUI”

---

## 🚀 使い方

1. タスクを入力
2. 緊急度・気分を選択
3. 「生成」ボタンを押す
4. **あなたのタスクは、華麗に正当化されます**
5. 気が向いたら作業しましょう（向かなかったらまた生成）
